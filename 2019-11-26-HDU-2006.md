### test-6

>[题目链接](http://acm.hdu.edu.cn/showproblem.php?pid=2006)

## 题意

给你n个整数，求他们中所有奇数的乘积。

## 代码  
```c
#include<cstdio>
using namespace std;
int main(){
	int a,n;
	while(scanf("%d",&n)!=EOF){
		long long   ans = 1;
		for(int i = 1 ;i <= n ;i++) {
			scanf("%d",&a);
			if(a%2==1)ans*=a;
		}
		printf("%lld\n",ans);
	}
return 0;
}
```
