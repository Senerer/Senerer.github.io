### test-6

>[题目链接](http://acm.hdu.edu.cn/showproblem.php?pid=2006)

## 题意

给你n个整数，求他们中所有奇数的乘积。

## 代码  
```c++
#include<cstdio>
#include<algorithm>
using namespace std;
int main(){
	int n,m;
	while(scanf("%d %d",&n,&m)!=EOF){
		long long   x = 0,y = 0;
		for(int a = min(n,m) ;a <= max(n,m) ;a++) {
			if(a%2==1)x+=a*a*a;
			else y+=a*a;
		}
		printf("%lld %lld\n",y,x);
	}
return 0;
}
```
## 小结
忘记m可能比n小的，读题问题
