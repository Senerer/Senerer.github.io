### test-3  

>[题目链接]http://acm.hdu.edu.cn/showproblem.php?pid=2002

## 题意

根据输入的半径值，计算球的体积。

## 代码  
```c
const double pi =3.1415927;
int main(){  
	double x;
	while(scanf("%lf",&x)!=EOF) {  	
	double  ans =4/3.0*x*x*x*pi;
	printf("%.3lf\n",ans);
    }  
}
```
